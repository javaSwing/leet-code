# 第6章 链表

在说链表(LinkedList)之前，我也看了其它几本书的数据结构讲解，有相当一部分是把数组和链表两种数据结构放在一起讲的。两都都是`线性数据结构`，都可以很方便的存储数据。但是数组在绝大多数的语言（Java、C#）中声明一个数组时，都要先声明
该数据的长度。而且在内存中来看，分配的是连续内存且大小是固定的。且在数据插入或者移除一个元素，成本都很高。因为需要移动元素。

链表存储有序的元素集合，但不同于数组。链表在内存中并不是连续的放置，且可以动态扩展。每个元素由自身和指向一个的引用的指针组合。如下图：


与数组相比较，链表元素的插入或者添加，并不需要移动其它元素，只需要找到目标位置，调整其它指向下个元素的引用，即可完成对应的操作。现实生活中一个很的例子就是火车。


## 单向链表

单向链表是指只有一个next指针，指向下一个节点的。单向串联链表结构.

类签名如下：

```typescript  title="LinkedList.ts"

export class LinkedNode<T> {
  val: T;
  next: undefined | LinkedNode<T>;
  constructor(v: T) {
    this.val = v;
    this.next = undefined;
  }
}

export interface LinkedListInterface<T> {
  /**
   * 添加一个节点
   * @param element
   * @returns
   */
  push: (element: T) => void;

  /**
   * 向特定位置插入一个元素
   * @param element
   * @param position
   * @returns
   */
  insert: (element: T, position: number) => boolean;

  /**
   * 获取指定位置的元素，无返回undefined
   * @param index
   * @returns
   */
  getElementAt: (index: number) => LinkedNode<T> | undefined;

  /**
   * 从链表中移除一个元素
   * @param element
   * @returns
   */
  remove: (element: T) => T | undefined;

  /**
   * 返回元素在链表中的索引
   * @param element
   * @returns
   */
  indexOf: (element: T) => number;

  /**
   * 从链表指定索引中移除一个元素
   * @param index
   * @returns
   */
  removeAt: (index: number) => T | undefined;

  isEmpty: () => boolean;

  size: () => number;

  toString: () => string;
}
```

### 实现

``` typescript title="具体实现"
export const defaultEquals = (a, b) => {
  return a === b;
};

export class LinkedNode<T> {
  val: T;
  next: undefined | LinkedNode<T>;
  constructor(v: T) {
    this.val = v;
    this.next = undefined;
  }
}

export default class LinkedList<T> implements LinkedListInterface<T> {
  count: number;
  head: undefined | LinkedNode<T>;
  eqFn: (a: T, b: T) => boolean;
  constructor(eq = defaultEquals) {
    this.count = 0;
    this.head = undefined;
    this.eqFn = eq;
  }

  push(element: T) {
    const node = new LinkedNode<T>(element);
    if (this.head == null) {
      this.head = node;
    } else {
      let current = this.head;
      while (current.next != null) {
        current = current.next;
      }
      current.next = node;
    }
    this.count++;
  }

  insert(element: T, position: number) {
    if (position >= 0 && position <= this.count) {
      const node = new LinkedNode(element);
      if (position == 0) {
        const c = this.head;
        node.next = c;
        this.head = node;
      } else {
        const pre = this.getElementAt(position - 1);
        if (pre) {
          const next = pre.next;
          pre.next = node;
          node.next = next;
        }
      }
      this.count++;
      return true;
    }
    return false;
  }
  getElementAt(index: number) {
    if (index >= 0 && index < this.count) {
      let current = this.head;
      for (let i = 0; i < this.count; i++) {
        if (i === index) {
          break;
        }
        current = current?.next;
      }
      return current;
    }
    return undefined;
  }

  remove(element: T) {
    const i = this.indexOf(element);
    return this.removeAt(i);
  }

  indexOf(element: T) {
    let current = this.head;
    for (let i = 0; i < this.count; i++) {
      if (current && this.eqFn(current.val, element)) {
        return i;
      }
      current = current?.next;
    }

    return -1;
  }

  removeAt(index: number) {
    if (index === 0) {
      if (typeof this.head !== 'undefined') {
        const c = this.head;
        this.head = this.head.next;
        this.count--;
        return c.val;
      }
    } else {
      const prev = this.getElementAt(index - 1);
      const current = prev?.next;
      if (prev) {
        prev.next = current?.next;
        this.count--;
        return current?.val;
      }
    }
    return undefined;
  }

  isEmpty() {
    return this.size() === 0;
  }
  size() {
    return this.count;
  }

  toString() {
    if (!this.head) {
      return '';
    }
    let result = `${this.head?.val}`;
    let current: LinkedNode<T> | undefined = this.head;
    for (let i = 1; i < this.count; i++) {
      current = current?.next;
      if (current) {
        result += `,${current.val}`;
      }
    }

    return result;
  }
}

```

## 双向链表

与单向链表相比较而言，双向链表在链表节点内部多了一个指针指向前一个节点。也就是说一个节点有`next`指针和`prev`指针。在遍历上，可以正向遍历也可以反向遍历。
