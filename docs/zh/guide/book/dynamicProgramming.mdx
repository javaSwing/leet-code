# 动态规划

动态规划（Dynamic Programming，简称DP），这个算法在我没有研究前。在刷LeetCode的时候，基本在评论或者题解区出现的频率特别的高。但是当时看完之后总是一头雾水，总是看到别人写的代码能理解，自己写完全没有思路。
而在本书中的这一块，写的也是比较的晦涩难懂，自己在微信读书上看到书友的评论也证明了这一点，在最少硬币数这个题目上书中的代码还有问题看下图。感谢书友们的提示，要不然真的以为是自己的错误。后面网上找一别一篇大神的[资料](https://labuladong.github.io/algo/di-ling-zh-bfe1b/dong-tai-g-1e688/)来看。
